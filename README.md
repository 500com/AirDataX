# AirDataX
基于淘宝早期开源的DataX二次开发

* 根据实际需要增加PostgreSql读写插件、Hadoop2.0支持等
* 通过添加Thrift方便 Python客户端调用
* 抽取公共配置到模板 方便任务配置
* 增加DataX On Yarn 提高资源利用率、并行任务执行速度
